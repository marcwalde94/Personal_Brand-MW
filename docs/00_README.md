## So nutzt Marc das Board
- Frage stellen: **"ask the board: <Frage>"** → 6 Advisors, Konflikte, Synthese, 1 nächster Schritt.
- Wichtige Entscheidungen landen in `06_Entscheidungsjournal.md` (Marc bittet: "log das").
- Das Board zieht IMMER `01`, `02`, `03` als Kontext und prüft `06` auf frühere Entscheidungen.

## Bewusste Designentscheidungen
- **6 Archetypen statt 8–12 reale Experten** (Marcs Wahl): schärfer, kein Namedropping, Relevanz-Filter pro Frage.
- **CEO-Modus ist Default**: Wahrheit vor Zustimmung, Denkfehler/Muster aktiv aufdecken.
- **Kein Content-Ingest realer Personen**: stattdessen stabile Denk-Frameworks in den Archetypen.
